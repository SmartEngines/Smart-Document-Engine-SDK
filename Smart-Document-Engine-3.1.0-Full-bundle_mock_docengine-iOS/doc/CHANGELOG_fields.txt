Field Naming Changes (Important - Update Your Code!)

Several fields have been renamed to improve consistency and clarity. Please review the full list (see end of this document) when updating your integrations! This includes changes like:

-   `inn` -> `inn_number`
-   `director` -> `director_signatory_name`
-   `accountant` -> `chief_accountant_signatory_name`
-   `beneficiary_bank` -> `beneficiary_bank_name` (and similar changes for several payment-related fields).
-   And many more - see the complete list below.
